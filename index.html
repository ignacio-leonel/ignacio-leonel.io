EL SUELDO DE JUAN

Título del Proyecto: "Análisis de Ganancias Mensuales de Juan"

Descripción: "En este proyecto, he realizado un análisis de las ganancias mensuales de Juan a lo largo de un año. El objetivo principal fue calcular el promedio de sus ganancias y determinar si su sueldo se considera alto, bajo o 'está bien' para cada mes. Para lograr esto, utilicé Python y las siguientes listas:

Lista de meses: Contiene los nombres de los meses del año, lo que facilita la identificación de las ganancias mensuales.

Lista de ganancias mensuales: Contiene las ganancias correspondientes a cada mes del año. Cálculo del Promedio:

Para calcular el promedio de las ganancias mensuales de Juan, diseñé una función llamada calcular_promedio que suma todas las ganancias y divide el total por la cantidad de meses. Este promedio se calcula y se muestra en pantalla.

Evaluación de los Sueldos: Luego de calcular el promedio, el programa evalúa cada mes individualmente. Utilicé un bucle for para recorrer ambas listas al mismo tiempo, lo que permitió mostrar las ganancias y la evaluación correspondiente para cada mes. La evaluación se basa en tres categorías: 'Bajo': Si el sueldo es inferior a $300. 'Está bien': Si el sueldo está entre $300 y $900. 'Alto': Si el sueldo supera los $900.

El programa imprime estas evaluaciones junto con las ganancias mensuales y los nombres de los meses.

Este proyecto muestra mis habilidades en programación con Python y mi capacidad para realizar análisis de datos básicos. Además, demuestra mi enfoque en la organización de datos y la resolución de problemas prácticos."
